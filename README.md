# AdventureWorks SQL Server & Power BI Analytics

This project demonstrates an end-to-end analytics solution using AdventureWorksDW, SQL Server and Power BI.

## Project scope

- Source data exploration in SQL Server
- Data modeling with SQL views/tables
- Star schema design
- Sales, Product & Inventory, Customer and Finance analysis
- Currency conversion logic
- Power BI semantic model and interactive dashboards

## Tech stack

- SQL Server
- Power BI
- DAX
- GitHub
